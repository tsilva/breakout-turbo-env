# Support

breakout-turbo-env supports CPython 3.11 or newer on:

- Apple-silicon macOS 11 or newer; and
- x86-64 Linux with glibc 2.28 or newer.

In short, the supported distributions are Apple-silicon macOS and x86-64 Linux.

Other operating-system and architecture combinations are unsupported. A source
archive is published for inspection and reproducible builds, but its presence
does not extend the supported platform set.

Use [GitHub Discussions](https://github.com/tsilva/breakout-turbo-env/discussions)
for installation help, usage questions, research ideas, and general feedback.
Use [GitHub Issues](https://github.com/tsilva/breakout-turbo-env/issues) for
reproducible bugs and scoped feature requests. Use the private process in
[`SECURITY.md`](SECURITY.md) for vulnerabilities.

When requesting help, include the package version, Python version, operating
system and architecture, installation command, and a minimal reproduction.
